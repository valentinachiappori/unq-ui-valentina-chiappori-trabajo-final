# Palabras Encadenadas - TFI

Trabajo Final Integrador — UNQ UI 2026.

## Instalación

Requisitos: Node.js (versión 20.19 o superior)

Clonar el repositorio e instalar las dependencias:

```bash
git clone https://github.com/valentinachiappori/unq-ui-valentina-chiappori-trabajo-final.git
cd unq-ui-valentina-chiappori-trabajo-final
npm install
```

## Ejecución local

```bash
npm run dev
```

Luego abrir `http://localhost:5173` en el navegador.
